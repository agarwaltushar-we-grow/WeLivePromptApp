# User Guide

## Start the app

Open **Prompt Polisher** from your desktop, Start menu, Applications folder, or launcher. You do not need VS Code, command prompt, Python, `.env`, or Ollama.

## Optimize a prompt

1. Paste your original prompt into **Original prompt**.
2. Add optional clarification details if needed.
3. Click **Analyze and optimize**.
4. Copy the English prompt, copy the Japanese translation, or export the result.

## If the app says the API key is not configured

Ask the developer/admin to open **Admin Settings**, log in, add the API key, and save settings. Normal users do not need to enter the key.
